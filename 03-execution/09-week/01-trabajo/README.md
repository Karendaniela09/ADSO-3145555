# Proyecto Base de Datos - Prueba Técnica

## Descripción
Este proyecto corresponde al análisis y estabilización de un modelo de base de datos en PostgreSQL.

## Contenido
- Análisis de dominios
- ADR (Architecture Decision Record)
- Backlog técnico
- Plan de datos de prueba
- Plan de trabajo inicial
- Estrategia de contenedorización y versionamiento

## Tecnologías
- PostgreSQL
- Liquibase
- Docker
